Mobile App Kaise Run Karein (Step-By-Step)?
Aap apne mobile app ko live test karne ke liye bas in tiny steps ko follow karein:

1. Terminal Open Kijiye aur mobile-app directory me jayein:
   cd mobile-app

2. Metro Bundler Start Karein (D Drive Optimization Runner):
   Bhai, aapki C Drive full hone ke karan direct "npx expo start" crash ho raha hai. Isko bypass karne ke liye maine "run-app.bat" banaya hai jo D Drive ka storage use karega.
   Simple isko run karein:
   .\run-app.bat

3. App Run Kaise Hoga?
   Android Physical Phone / Emulator: Terminal me "a" dabayein ya Expo Go scan karein.
   iOS Physical Phone / Simulator: Terminal me "i" dabayein ya Expo Go scan karein.
